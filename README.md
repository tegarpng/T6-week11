## TUGAS PEMVIS REST API
Tugas ini meminta membuat aplikasi desktop Python (PySide6) yang bisa mengelola data post dari REST API nyata.
Intinya adalah aplikasi CRUD lengkap dengan 5 operasi:

1. GET — tampilkan semua post dalam tabel
2. POST — tambah post baru lewat form
3. PUT — edit post yang dipilih
4. DELETE — hapus post dengan konfirmasi (comments ikut terhapus)
